구조체(structure)
	데이터의 집합, 멤버 함수 없음
	여러 자료형을 가진 변수들을 하나의 패키지로 만들어 놓은 것
	C에서는 모든 자료형이 값형
		하지만 주소를 전달하면 참조형처럼 쓸 수 있음
		즉, 주소를 전달하지 않는 한 값형
	구조체의 필요성
		사람은 보통 물체 단위로 봄
		실수도 막을 수 있음
	초기화x
	typedef
		이미 있는 자료형에 새로운 별명을 지어줌
		1.
			struct date{
				...
			};
			typedef date date_t;
		2.
			typedef struct date{
				...
			}date_t;
		3.
			typedef struct{
				...
			}date_t;
		enum도 마찬가지
	가능하면 구조체, 열거형, 공용체에 typedef를 써서 제대로 된 자료형처럼 보이게 하는게 좋음
	구조체는 선언과 동시에 초기화 안됨
		스택에 남아있는 데이터를 그대로 사용
		구조체는 지역 변수를 여러 개를 따로따로 사용하는 것과 마찬가지
			기계는 구조체 개념을 모름
	배열 초기화 떄 썼던 {0, }로 초기화 가능
	컴파일러에 따라 구조체가 차지하는 모든 메모리 공간을 0으로 채워주는 명령어로 바꿔줄 수도 있음
		memset() //나중에...
	요소 나열법으로도 초기화 가능
		실수할 수 있으니 사용x
		const 멤버 변수 초기화할 때는 유용함
			하지만, const 멤버 변수는 잘 안씀
